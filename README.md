This project is a simple calculator developed in Python that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. The program allows users to enter two numbers and select an operation, after which it calculates and displays the result.

The calculator is designed to be easy to use and serves as a beginner-friendly project for learning Python programming. It demonstrates fundamental concepts such as user input, conditional statements, arithmetic operators, and basic program flow.

This project is ideal for students and beginners who want to practice Python basics while building a useful command-line application.
